BlogSphere - Modern Blogging Platform

A full-stack blogging application frontend built using React and modern web technologies. BlogSphere provides a clean, responsive, and user-friendly platform for reading, writing, editing, and managing blog articles with secure authentication and protected routing.

# Project Accomplishments
Built the frontend using React + Vite.
Implemented dynamic client-side routing using React Router.
Created reusable UI components for scalability.
Developed authentication state management using store architecture.
Added protected routes for authorized users.
Integrated Axios configuration for API communication.
Designed responsive layouts for all screen sizes.
Organized the project into modular folders for better maintainability.
# Core Concepts & Implementation
1. Component-Based Architecture (React.js)
Definition: Building UI using reusable independent components.
Purpose: Used to create reusable pages like Login, Register, Home, Profile, and Article Forms.
2. Client-Side Routing (React Router)
Definition: Navigation between pages without full page reloads.
Purpose: Used for smooth transitions between authentication pages, articles, and profiles.
3. Authentication State Management
Definition: Managing user login state globally across the application.
Purpose: Used to protect routes and maintain user sessions efficiently.
4. API Integration (Axios)
Definition: Handling HTTP requests between frontend and backend.
Purpose: Used to fetch articles, manage authentication, and perform CRUD operations.
